# Скриншоти для presentation.html

Покладіть сюди реальні скриншоти UI — слайд «Що користувач бачить одразу»
(`presentation.html`) підхопить їх автоматично. Поки файлу немає, у рамці
показується плейсхолдер.

Очікувані імена (16:10, бажано ≥1280px завширшки):

| Файл             | Екран               |
| ---------------- | ------------------- |
| `login.png`      | Вхід / реєстрація   |
| `account.png`    | Сторінка акаунта    |
| `admin.png`      | Адмін-панель        |
| `support.png`    | Чат підтримки (AI)  |
| `playground.png` | API playground      |
